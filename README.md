# gra-panstwa-miasta
Aby odpowiedź była poprawna musi zaczynać się z wielkiej litery, a cała reszta dopowiedzi musi być z małych liter.<br/><br/>
Kopilacja serwera
```cmake .```
, następnie
```make``` <br/>
lub <br/>
```g++ --std=c++20 -Wall -O0 -g -pthread -o server server.cpp```

Włączenie servera 
```./server [port]```

Włączenie klienta
```python3 client.py [ip][port]```
